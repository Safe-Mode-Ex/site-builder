# Инструкция по запуску и началу работы

Чтобы начать работать с React и пакетами npm, необходимо установить [Node.js](https://nodejs.org/en/download).

## Установить зависимости

```npm i```

## Запустить в режиме разработки

```npm start```

## Собрать prod

```npm run build```

## Запустить prod

```npm run preview```

## Проверить линтером

```npm run lint```
